# FH6 Rims/Appearance Locked Tune Editor

Ever download a fast tune, but the build is absolutely awful looking? This tools solves this problem by editing the locally saved tune files in your gamesave folder. 
This does NOT contain any code for reading, editing, or unlocking tune files in full form. This app is for changing cosmetics only.

# How to use

### Download/updating
Download the tool from the Releases tab. You will get a .zip containing everything needed to use the tool. If an update is needed, the tool will automatically install it in a folder with the new version number (and uninstall the old one). 
The gamesave folder should be auto-detected (Xbox gamepass only for now) but can be set manually if you have a different save location (Steam, onlinefix, etc). You should now see all your downloaded/local tunes ready to be edited.

### Modifying a tune
After you pick a tune, you can edit any available tire and rims/aero and appearance upgrades. All slots update based on car/bodykit, exactly as it does in-game, and shows the real part list + names. This ensures you never build an invalid tune.
Once you finish making your changes, click save. You will get the option to either: Overwrite the tune, or create a new copy with your changes. Refresh/re-open the tune manager and find your tune. Install and enjoy!

### Making presets
You can make upgrade presets for each car to speed up tune editing. This is per car, so each preset has an extension based on car ordinal. This makes it so you don't install invalid tune values accidentally. 
After editing a tune, you will see a diff map of what you've changed. These are the changes that will apply to other tunes, so if you only change the Front Bumper, the preset will only change the Front Bumper when applied.
