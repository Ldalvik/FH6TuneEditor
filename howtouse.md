### Modifying a tune

1. After you pick a tune, you can edit any available tire and rims/aero and appearance upgrades. All slots update based on car/bodykit, exactly as it does in-game, and shows the real part list + names. This ensures you never build an invalid tune.
2. Once you finish making your changes, click save. You will get the option to either: Overwrite the tune, or create a new copy with your changes. Refresh/re-open the tune manager and find your tune. Install and enjoy!

### Making presets
You can make upgrade presets for each car to speed up tune editing. This is per car, so each preset has an extension based on car ordinal. This makes it so you don't install invalid tune values accidentally. 
After editing a tune, you will see a diff map of what you've changed. These are the changes that will apply to other tunes, so if you only change the Front Bumper, the preset will only change the Front Bumper when applied.
